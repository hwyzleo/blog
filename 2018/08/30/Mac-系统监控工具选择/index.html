Mac 系统监控工具选择

发表于 2018-08-30 | 更新于 2018-09-04 | 分类于 Mac软件 | 阅读次数：

终于入坑Mac了，个人感觉系统监控类APP装一个挂在菜单栏里对自己系统有个实时了解还是挺有必要的，尤其是CPU、上传下载速率。这类应用貌似可选择面不多，推荐的都集中在2、3个APP上，正好借此机会比较下各自的区别。

快速入口：->全功能比较点此　->综合结论点此

应用介绍

iStat Menus

支持正版　囊中羞涩


iStat Menus应该是首推的APP了吧，从各类文章推荐数及各类网站的下载数量来看，都是实至名归。

iStatistica

支持正版　囊中羞涩


iStatistica也是不少人推荐的一款APP，界面也非常美观

System Monitor

支持正版　囊中羞涩


这款貌似用的不多，功能还算挺全，但也不便宜，界面上过于单调，难以吸引人

Instastats

支持正版

这个外观挺不错的，改天测试一下，突然发现其他网站15年介绍它的时候这款APP还是免费的……，现在竟然收费了，关键是之后就没更新过了，太不厚道了……

话说就没有免费好用的系统监控工具吗？

附加功能

蓝牙

iStatistica


此功能能显示已连接蓝牙设备，个人感觉没啥用

远程

iStatistica

该功能支持远程查看本机监控数据，但感觉个人使用的话意义不大

完整性

iStat Menus

  • 3级布局：总览 -> 子项 -> 详情，层层递进，非常直观

iStatistica

  • 1级布局：基本内容满足

System Monitor

易用性

iStat Menus

菜单栏直接显示，点击分3级显示，非常方便

iStatistica

菜单栏直接显示，点击具体具体内容，非常方便

System Monitor

菜单栏直接显示，点击具体具体内容，非常方便

定制性

iStat Menus


iStat Menus定制性非常高，各项设置允许单独的快捷键，允许调整显示项的颜色、顺序、显示方式、数量、图标、位置等

iStatistica


iStatistica定制性较低，只允许显示/隐藏具体监控项以及调整显示顺序

System Monitor


System Monitor定制性不错，允许显示/隐藏监控项及子内容，支持调整图标大小，顺序，显示内容

综合结论

  • 愿意花钱的话iStat Menus是首选，界面美观，功能齐全，定制性强，简直已经做到监控类APP的极致了
  • 注重性能的话System Monitor也不错，功能足够，应用不大，内存占用小，就是美观度稍有欠缺
